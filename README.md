# Mercado-App
Ao iniciar o programa, será exibido um menu principal com as opções Produtos, Clientes, Realizar Compra, Controle de Estoque e Sair. O usuário deve digitar o número correspondente à operação desejada.

## Produtos

### Como cadastrar um produto?

Acesse a opção Produtos e selecione a função de cadastro. Informe os dados solicitados pelo sistema, como código, nome, preço e quantidade em estoque. Após a confirmação, o produto será armazenado para futuras consultas e vendas.

### Como consultar produtos?

Selecione a opção de listagem ou busca de produtos. O sistema exibirá os produtos cadastrados e suas informações, permitindo localizar itens específicos quando necessário.

### Como alterar um produto?

Escolha a opção de edição, informe o código do produto desejado e altere os dados necessários. Após salvar as alterações, as informações serão atualizadas no sistema.

### Como remover um produto?

Selecione a opção de exclusão e informe o código do produto. O sistema localizará o item e realizará sua remoção do cadastro.

## Clientes

### Como cadastrar um cliente?

Acesse o menu Clientes e escolha a opção de cadastro. Informe os dados solicitados, como CPF, nome, telefone e e-mail. Após a confirmação, o cliente ficará disponível para futuras compras.

### Como consultar clientes?

Selecione a opção de consulta para visualizar os clientes cadastrados e suas informações.

### Como alterar os dados de um cliente?

Escolha a opção de edição, informe o CPF do cliente e atualize os dados desejados. As alterações serão registradas imediatamente.

### Como excluir um cliente?

Selecione a opção de remoção e informe o CPF do cliente. O cadastro será excluído do sistema.

## Realizar Compra

### Como iniciar uma compra?

Selecione a opção Realizar Compra no menu principal. O sistema verificará se o cliente já possui cadastro.

### O que fazer se o cliente não estiver cadastrado?

Será possível realizar um cadastro rápido informando os dados necessários para prosseguir com a compra.

### Como adicionar produtos à compra?

Informe o código do produto desejado e a quantidade que será comprada. O sistema verificará automaticamente se há estoque suficiente.

### Como finalizar a compra?

Após adicionar todos os produtos, confirme a finalização da venda. O sistema calculará o valor total e gerará uma nota fiscal contendo os dados do cliente, os produtos comprados, as quantidades e os valores da compra.

## Controle de Estoque

### Como consultar o estoque?

Acesse a opção Controle de Estoque para visualizar a quantidade disponível de cada produto cadastrado.

### Como o estoque é atualizado?

Sempre que uma compra é concluída, o sistema reduz automaticamente a quantidade dos produtos vendidos, mantendo os dados atualizados.

### Como verificar a disponibilidade de um produto?

Ao consultar o estoque, será possível visualizar a quantidade restante de cada item e identificar quais produtos precisam de reposição.

## Sair

### Como encerrar o sistema?

Selecione a opção Sair no menu principal. O programa será finalizado e todas as informações salvas permanecerão armazenadas para uso futuro.
